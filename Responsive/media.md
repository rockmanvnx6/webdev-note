# Media Queries

```css
@media screen and (max-width: 1000px) {
	 /* rule for max screen 1000px */
}

```

## Common break point:

```css
@media only screen and (min-width: 768px) {
    /* tablets and desktop */
}

@media only screen and (max-width: 767px) {
    /* phones */
}

@media only screen and (max-width: 767px) and (orientation: portrait) {
    /* portrait phones */
}
```
