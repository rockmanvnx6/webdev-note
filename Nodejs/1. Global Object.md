# Global object

## To run nodeJS application

```bash
node filename
```

or 

```bash
node filename.js
```



## Global object

### Current dirrectory:

`__dirname`

- Print out the current directory:

- Example:

  ```javascript
  console.log(__dirname);
  ```

### Current file

`__filename`

- Print out the current filename:

- Example:

  ```javascript
  console.log(__filename);
  ```
