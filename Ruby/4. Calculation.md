# Calculation

```ruby
puts "1+1 = #{1+1}"
```

Or

```ruby
puts ("1+1" + (1+1).to_s)
```

**abs()**

get absolute value

```ruby
num = -20
puts num.abs()
```

> 20

**round()**

20.5 => 21

20.4 => 20

```ruby
num.round()
```

**ceil()**

20.1 => 21

```ruby
num.ceil()
```

**floor()**

```ruby
num.floor()
```

20.9 => 20.
